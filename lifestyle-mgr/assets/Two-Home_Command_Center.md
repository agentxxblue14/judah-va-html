### How I run it
- One status language everywhere: **Active / Hold / To source / Return / Done.**
- Anything in **Hold** or **To source** gets a vendor scorecard run before it's booked (see the Vendor Vetting Scorecard sample).
- Every Friday you get a one-screen digest: what got done, what's pending, what needs a decision, and next week's plan. You only weigh in on the decisions.
