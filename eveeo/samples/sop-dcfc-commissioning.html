SOP 04: DC Fast Charging Station Commissioning & Energization

Owner: Strategic PM · Applies to: all DCFC site go-lives · Version: sample v1
This is a sample I built to show the format and rigor I would bring to EVEEO's SOP library. It uses standard DCFC commissioning practice. On the job, every step would be calibrated to EVEEO's actual hardware vendors, utility partners, and historical lessons learned.

1. Purpose

Ensure every DC Fast Charging station is safely energized, tested, and handed to operations in a repeatable way, so any qualified technician can run a go-live without tribal knowledge and nothing reaches the client untested.

2. Roles & responsibilities

RoleOwns
Strategic PMReadiness gate, sign-off, exception calls
Lead electricianInterconnection, torque checks, insulation tests
Commissioning techCharger config, network registration, test sessions
Site host contactAccess, parking control, final walkthrough

3. Pre-energization gate (do not proceed until all true)

  • Utility interconnection approved and meter set permit on file
  • As-built matches approved electrical drawings
  • Transformer and switchgear inspected, torque-marked
  • Equipment grounding verified and recorded
  • ADA stall dimensions, signage, and clearances confirmed
  • Firmware version logged for each dispenser

4. Energization sequence

  1. Confirm lockout/tagout removed only by the authorized lead electrician.
  2. Energize upstream, verify voltage and phase rotation against spec.
  3. Power each dispenser in sequence; confirm boot, no fault codes.
  4. Register each unit to the network management system; confirm online status.
  5. Set pricing, access mode, and load-management profile per the site plan.

5. Commissioning tests (record pass/fail per unit)

TestMethodPass criteria
Insulation resistanceMegohmmeter on de-energized circuitWithin manufacturer threshold
Live charge sessionTest EV or load box, full handshakeReaches rated power, clean stop
Emergency stopTrigger E-stop mid-sessionImmediate safe shutdown
Payment / activationRun app and RFID startBoth start and bill correctly
Network reportingConfirm session in dashboardSession and kWh logged

6. Handover to operations

  • All units online, zero open faults for 24 hours
  • Test sessions and readings filed to the project record
  • Site host walkthrough complete, signage and access confirmed
  • Maintenance schedule and warranty contacts entered into the tracker
  • PM final sign-off; project moved to "Operational"

7. Exceptions

Any failed gate item or test stops the go-live. The commissioning tech logs it, the PM decides remediate-and-retest or reschedule. No station is handed over with an open exception.
